总结：

对比两个 header 字段：我比较浅显的做了下区别就是：
一个是：允许请求头中携带，另一个是：允许访问携带的该属性。
就像客人来了家里做客，
一个是：我允许你来我家，一个是：我允许你来我家里吃饭、睡觉 …
