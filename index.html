Git 版本控制

初始化

  1. git init 用来初始化本地 Git 仓库，可进行一系列 git 操作
  2. git init –bare 用来初始化远程 Git 裸仓库，用来记录提交的修改信息 ，不可进行一系列 git 操作

工作流-暂存区回滚

  1. git add . 提交到暂存区
  2. git reset HEAD text.name 清空暂存区的修改
  3. git checkout – text.name 工作区回滚到修改之前
